# Udemy Microservices Project

This repository contains a microservices-based e-commerce project built with .NET.

## Technologies

- .NET
- Docker & Docker Compose
- RabbitMQ
- Clean Architecture
- CQRS & MediatR
- Entity Framework Core
- MongoDB / Redis / PostgreSQL / SQL Server

## Purpose

I am building this project step by step to improve my backend development, microservices architecture, and distributed systems skills.
